## 블로그를 만든 이유
- `Next.js`를 활용해서 블로그를 만든 경우를 많이 봤고 저도 직접 만들어보고 싶었습니다!
- App router 적용, Tailwind CSS, shadcn/ui등 최신 기술을 적용해보고 싶었습니다.
- 꾸준히 develop 해가며 저의 색깔이 많이 묻는 블로그를 만들어보고 싶었습니다.
